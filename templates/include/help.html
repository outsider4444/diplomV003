Справка

О программе

Название веб-приложения: "Учет деталей"

Предназначение: Данная программа создана для учета данных о изделиях, материалах, сотрудниках, заказчиках, поставщиках, нарядах и ОТК

Поводом для создания данного веб-приложения является дипломная работа.

Версия: 1.0

Справка о функционале

Возможности:
  • Добавление новых сотрудников
  • Просмотр информацию о существующих сотрудниках
  • Редактирование информации о сотрудниках
  • Удаление сотрудников
  • Поиск по ФИО
  • Сортировка по коду и ФИО
  • Фильтрация по должностям
Функции работы на странице "Сотрудники":
  1. Просмотр данных:
    1. Нажмите на вкладку "Сотрудники". Произойдет переход на страницу сотрудников.
    2. В таблице представлен список всех сотрудников, находящихся в базе данных
    3. Введите ФИО либо часть ФИО сотрудника в поле для поиска. Если введенные данные присутсвуют в базе данных, то выведутся все сотрудники соотвествующие данному запросу.
    4. Фильтрация данных по должностям происходит путем выбора пункта в меню фильтрации. После нажатия на "Поиск", происходит вывод всех сотрудников с соотвествующей должностью.
    5. Сортировка данных происходит путем выбора метода сортировки в верхней части таблицы. После выбора пункта автоматически происхиодит сортировка данных.
    6. По нажатию на код, ФИО или должность сотрудника, произойдет переход на страницу с подробным описанием данного сотрудника
    7. На странице с подробным описанием сотрудника находится вся информация о сотруднике, хранящаяся в базе данных.
  2. Ввод новых данных
    1. Для создания нового сотрудника необходимо перейти на страницу сотрудников, выбрав соответсвующий пункт в шапке сайта
    2. Нажмите на кнопку "Добавить нового сотрудника". Произойдет переход на страницу добавления нового сотрудника.
    3. Заполните форму и нажмите на кнопку "Подтвердить". Если данные введены корректно то сотрудник добавится в базу данных. Если нет, то появится сообщение о соответсвующей ошибке
  3. Редактирование
    1. Для редактирования информации уже существующего сотрудника в базе данных, необходимо перейти на страницу сотрудников, выбрав соответсвующий пункт в шапке сайта
    2. Выберите необходимого сотрудника на странице со списком сотрудников и перейти на форму с подробным его описанием
    3. Выберите пункт "Редактировать сотрудника"
    4. После перехода на форму, вы можете изменить информацию о сотруднике
    5. Нажав кнопку "Подтвердить", произойдет обновление данных о сотруднике
  4. Удаление сотрудника
    1. Для удаления информации уже существующего сотрудника в базе данных, необходимо перейти на страницу сотрудников, выбрав соответсвующий пункт в шапке сайта
    2. Выберите соответствующего сотрудника на странице со списком сотрудников и перейти на форму с подробным его описанием
    3. Выберите пункт "Удалить сотрудника"
    4. После перехода на форму удаления, подтвердить данное действие, нажатием на кнопку "Удалить сотрудника"

  3. Редактирование
    1. Для редактирования информации уже существующего изделия в базе данных, необходимо перейти на страницу изделий, выбрав соответсвующий пункт в шапке сайта

    5. Нажав кнопку "Подтвердить", произойдет обновление данных о сотруднике

    1. Для добавления/редактирования информации о форме изделия, необходимо перейти на страницу изделий, выбрав соответсвующий пункт в шапке сайта
    2. Выберите соответствующее изделие на странице со списком изделий и перейти на форму с подробным его описанием
    3. Нажав на кнопку "Показать/Скрыть дополнительную информацию"
    4. В появившейся форме выберите пункт "Добавить форму изделия", в случае если форма отсутсвует
    5. Произойдет переход на форму с добавлением информацию о форме изделия
  5. Добавление нового месяца в календарь
    1. Для добавления/редактирования информации о форме изделия, необходимо перейти на страницу изделий, выбрав соответсвующий пункт в шапке сайта
    2. Выберите необходимое изделие на странице со списком изделий и перейти на форму с подробным его описанием
    3. Нажав на кнопку "Показать/Скрыть дополнительную информацию"
    4. Нажав на кнопку "Показать/Скрыть добавление даты в календарь"
    5. Заполните форму и нажмите "Подтвердить"
  6. Удаление изделия
    1. Для удаления информации уже существующего изделия в базе данных, необходимо перейти на страницу изделий, выбрав соответсвующий пункт в шапке сайта
